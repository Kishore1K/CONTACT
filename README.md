# CONTACT FORM

IF U WANT TO RUN THIS 
<br/>
 *STEPS*
 1. CLONE 
  ```shell
    $ git clone https://github.com/Kishore1K/CONTACT.git
 
 ```
 2. CHANGE TO THE DIR
  ```shell
    $ cd /CONTACT 
   ```
 3. INSTALL ALL THE NODE FILES  
 ``` shell 
 $ npm install
 ``` 
 
 4. RUN THE FILES 
 ```shell 
 $ npm start
 ```
